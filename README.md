# IoT_project
This is a private repository related to a university project on the IoT. 
More specifically, this repository contains code for a speed camera and more.


## Overview
This project implements a scaled-down speed detection system (autovelox) using:
 *ESP32* with two infrared sensors for speed measurement.
 *Arduino UNO* on a small car as an infrared receiver.
 *Web dashboard* for live monitoring.
