IoT Speed Camera Project

Embedded Software for the Internet of Things

Project Overview

The Speed Camera project is an educational system designed to simulate an intelligent speed detection device.
It uses an ESP32 board equipped with two infrared sensors to measure the time taken by the vehicle to move from the first to the second signal, then calculates the speed and illuminates the corresponding LED, saving the data.
The car part uses an Arduino UNO board equipped with an infrared receiver which, when it picks up a signal, alerts a hypothetical driver with a dedicated sound emitter and LED.
The project integrates hardware sensors, wired communication and web-based visualisation in an IoT context.

System Architecture

ESP32 (Autovelox Unit)

  • Reads IR sensors for speed measurement
  • Calculates velocity and detects overspeed
  • Hosts a web server showing speed logs
  • Sends IR beacon to Arduino car

Arduino Car Receiver

  • Receives IR beacon from ESP32
  • Activates LED or buzzer alert
  • Simulates driver notification system
